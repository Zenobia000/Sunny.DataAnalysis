[周刊王CTWANT] 綠營緊咬張善政在企業任職時，代表爭取農委會研究案，也涉嫌報告抄襲。對此，中國國民黨文傳會副主委林家興回應指出，黨中央已經針對指控內容與事實真相進行瞭解，只能說整件事就是「張飛打岳飛」，因為張善政被指控的是民間企業「宏碁」當年受農委會委託的計劃案，共有12個人的研究團隊作業，當時張善政是宏碁的部門主管，所以「掛主持人」，並非張善政一個人獨力執行與撰寫，這份委託研究報告的情況，無法比擬個人正式學術學位論文涉嫌抄襲。

林家興說，該委託案長達三年，過程橫跨藍綠兩屆不同黨派政府，不只是寫報告，還要蒐集各國資料實地考察、建立實驗機制、辦研討會、座談會、跨部會溝通、各國考察、發行電子報、系統開發、農委會相關計畫輔導等，國民黨檢視網路研究案資料後，認定成果豐碩，並非只有媒體所稱的「6頁書面報告」，實際上不同份數報告分別有上千頁，包括97年度就製作超過11份專題子報告，以及國際資訊應用策略分析、實地參訪心得等等，相關資料全數在相關網站上公開都能查閱，歡迎社會公評。
